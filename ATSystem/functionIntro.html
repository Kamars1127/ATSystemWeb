檔案 問事表 聯絡資訊 問事紀錄 關於

新增問事表

功能 : 用來建立問事表

日期與地點 : 組合成問事表的名稱

問事表名稱不能重複，被建立

瀏覽問事表

功能 : 瀏覽已儲存的問事表

查詢 & 輸入框 :

1. 可以依地點或日期查詢問事表。

2. 日期只能用純數字。(2021年3月29日 就輸入 20210329)

3. 只能輸入文字(英文、中文)和數字。

顯示全部 : 顯示全部問事表

點選表格列會顯示工具視窗

1. 對選取得的資料表作開啟、刪除、匯出PDF。

2. 開啟 : 在問事表功能選單中開啟。 (2021年3月29日 就輸入 20210329)

3. 匯出PDF : 在電腦上儲存PDF檔。

4. 刪除 : 把整個問事表刪除，包含裏頭的問事內容(請三思)

新增問事地點

功能 : 新增問事地點

編輯問事地點

功能 : 對存在的問事地點做修改或刪除

查詢 & 輸入框 :

1. 查詢已存在的地點。

2. 只能輸入文字或數字。

全部 : 顯示全部地點。

修改 : 修改地點名稱。

刪除 : 刪除該地點。 (如果該地點被用來建立問事表則無法刪除)

報名

功能 : 建立報名資訊或聯絡資訊(新人)

新人 :

1. 姓名和居住地不能空白。

2. 建立報名資訊同時，也會建立聯絡資訊。(不必再去聯絡資訊功能選單新增)

舊人 :

1. 輸入姓名後，可以在表格中選取正確的問事人。

2. 不用輸入完整姓名，只要輸入關鍵字就可以在表格中選取正確問事人。

3. 同一個人，不能重複報名。

報名列區

功能 : 查看報名資訊，刪除報名，新增問事內容

檢視 : 查看報名資訊

問事 :

1. 新增問事內容。

2. 聯絡資訊 : 查看問事人的聯絡資訊。

3. 歷史紀錄 : 問事人過往問事資料。

4. 備註 : 報名時的備註內容，這邊可以更改。

5. 新增後，會直接存到資料庫。

問事區

功能 : 編輯問事內容，或刪除問事內容

新增聯絡資訊

功能 : 建立聯絡資訊

注意 : 姓名和居住地不能空白

其他

查詢 : 可以輸入姓名、日期(數字)、居住地、電話

顯示全部 : 顯示全部的聯絡資料

檢視完整資訊 :

1. 查看個人完整的聯絡資訊。

2. 可以查看個人全部的問事記錄。

3. 可以更新姓名、聯絡電話、居住地、備註。

3. 也可以刪除此人的聯絡資訊。(問事資料也會一併刪除，等同於刪除帳號)

刪除 : 等同於檢視完整資訊中的刪除

查詢指定對象的紀錄

必須輸入完整姓名，才能做查詢

同名同姓時，需要確認查詢的對象是誰

完整內容 : 可以查看該筆紀錄詳細內容

顯示紀錄的完整內容

查看該筆紀錄詳細內容
